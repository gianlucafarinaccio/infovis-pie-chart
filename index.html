Interactive Pie Chart

Create a JSON file with trivariate data: there are 10 data-cases, and each data-case has three quantitative variables whose values are all positive. Draw this dataset using a pie chart where the first quantitative variable is used to determine the angle of each pie slice (note: you need to create a proportion so that the total sum corresponds to 360Â°). When clicking with the left mouse button anywhere on the chart, the second variable is used instead of the first for all pie slices. Clicking again switches to the third variable. On the third click, it switches back to using the first variable, and so on, cycling through the three. Make sure the transitions are smooth and not abrupt. Use D3.js scales to map the range of the variable values (which can be arbitrary) to the range of visual dimensions (which depends on your interface).
